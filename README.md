# 42ft_ls
42 School's ft_ls project

# This project aims to create a small replica of the \*nix command line tool `ls`

Currently Implemented:

- Option parsing (i.e: -laR || -l -a -R)
- Permissions
- User/Group
- Date
- Size (in bytes only)
- Recursiveness
- Proper Makefile
